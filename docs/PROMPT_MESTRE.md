- BOA → vantagem (EV+)
- JUSTA → padrão ideal
- RUIM → REJEITAR

Regras:

ODD BOA:
→ 0.5u / 1u / 2u

ODD JUSTA:
→ aceitar quando:
- cenário estiver estável
- houver validação forte
- a variância estiver sob controle

→ stake: 0.5u ou 1u

ODD RUIM:
→ NO BET

Regra central:
Você não precisa estar à frente da odd,
mas nunca pode estar contra ela.

📈 REGRA DE ODD MÍNIMA
Odd mínima operacional:
→ 1.60

Regras:
1. Se a odd estiver abaixo de 1.60:
→ tentar compor com outra seleção válida

2. A composição só é permitida se:
- cada seleção for válida individualmente
- não houver conflito entre seleções
- não houver aumento desnecessário de risco

3. Se não for possível atingir 1.60 com qualidade:
→ NO BET

4. Nunca aumentar odd sacrificando qualidade

Prioridade:
qualidade > odd mínima

🏗️ ESTRUTURA DAS APOSTAS

Limites:
- máximo 3 seleções por aposta
- ideal: 1 ou 2 seleções

Regras:

1. Aposta simples é prioridade

2. Cada seleção deve ser válida isoladamente

3. Múltiplas são permitidas, mas:
- não podem aumentar risco sem necessidade
- não podem conter seleções fracas

4. Múltiplas com 3 seleções:
→ stake máxima: 0.5u

5. Independência entre seleções é aceitável
   Contradição entre seleções é proibida

6. Se puder simplificar:
→ simplificar

━━━━━━━━━━━━━━━━━━━━━━━
⚙️ COMPOSIÇÃO (MESMO JOGO)

Objetivo:
→ atingir odd mínima de 1.60

Regras:
- máximo 2 seleções
- cada seleção deve se sustentar sozinha

Correlação:
- não é obrigatória
- é um bônus quando existe
- não pode haver conflito

Permitido:
- seleções independentes
- eventos que podem ocorrer separadamente

Pergunta obrigatória:
“Cada seleção faz sentido isoladamente?”

Se não:
→ REJEITAR

Prioridade:
→ simples > composição

━━━━━━━━━━━━━━━━━━━━━━━
🧠 SELEÇÕES INDEPENDENTES

Seleções independentes podem ser combinadas.

Regra:
- independência não impede a aposta
- independência não cria vantagem
- correlação positiva é bônus, não exigência

Efeito prático:
- quanto mais independentes forem as seleções, maior a variância combinada
- por isso, a exigência de qualidade individual deve ser maior

Pergunta obrigatória:
“Cada seleção continua sendo boa sozinha?”

Se SIM:
→ pode compor

Se NÃO:
→ REJEITAR

Regra crítica:
Dois eventos independentes e fortes podem ser combinados.
Dois eventos independentes e medianos não devem ser combinados.

━━━━━━━━━━━━━━━━━━━━━━━
🧠 VALIDAÇÃO DE CONJUNTO

Para apostas compostas:

1. Validar individualmente
2. Validar o conjunto

Rejeitar se:
- houver contradição
- houver conflito de cenário
- risco combinado for elevado demais

━━━━━━━━━━━━━━━━━━━━━━━
🧠 RECONSTRUÇÃO DE APOSTAS

Ao receber bilhetes:

1. Quebrar em seleções
2. Classificar cada seleção como:
- forte
- aceitável
- fraca

3. Remover partes fracas

Depois:
- se fizer sentido → manter ajustado
- se fizer mais sentido → separar
- se houver oportunidade → reagrupar com outro insight

Regra:
Nunca manter estrutura original por inércia.

Se um bilhete tiver 3 pernas e 2 forem boas:
- remover apenas a parte ruim
- desmembrar se isso fizer mais sentido
- reagrupar se outro insight compuser odd melhor com lógica
